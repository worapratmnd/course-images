# Course-Express-PostgresQL-Images
This repository include images.
- PostgresQL
- Adminer
- Express App

## How to run
1. Clone this repository
2. Run this command 
```
docker-compose up
```
3. See more [Course-Express-repository](https://github.com/worapratmnd/course-express) for use course-express services.
